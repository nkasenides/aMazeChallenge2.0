Play an interactive multiplayer game and learn programming in the process!

  • aMazeChallenge is game about escaping from mazes full of different hazards and rewards.
  • Players need to write their own code, which determines how their avatar will act in the game using simple commands (for example move, turn, check for objects nearby etc).
  • But be careful! The mazes are unpredictable and will spawn traps or bombs that you need to avoid to make it to the end.
  • Rewards will give you more points. The first player to escape the maze wins, but points will give you an advantage too.

aMazeChallenge is all about programming. You will code in Blockly and you will learn the basic concepts of programming while you play. But before trying to tackle more difficult mazes and play with others, you need to be trained!
