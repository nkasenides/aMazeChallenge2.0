Writing your code (1/3)

The functions

There are two functions you need to use in your code:

Initialize function

The initialize function runs its instructions only once, before anything else is executed. It is used to declare variables that will be used in the rest of the code.

Run function

The run function runs its instructions continuously (at every turn). It is used to contain the algorithm that will move the player inside the maze and allow them to reach the end.

Useful hints

  • Do not place code outside the initialize and run functions. It will not be executed.
  • Try to include both initialize and run functions in your code. The run function is mandatory but initialize can be omitted, although better code will need to use both.
  • Do not declare any variables inside the run function. This will reset their values each time the algorithm is executed. Declare them in initialize instead.
  • Each block's structure indicates what its output or input is. If something does not fit, it was not meant to fit.
  • Saving your code after you have made some changes will avoid any accidental losses.

Moving & Turning

Your avatar can perform one of the following actions at each turn. When one of these commands is executed, your turn is completed.

Move Forward

This will advance your avatar one step forward in the direction it is facing.

Turn Clockwise

Your avatar will turn clockwise (to its right hand side), according to the direction it is facing.

Turn Counter-Clockwise

Your avatar will turn counter-clockwise (to its left hand side), according to the direction it is facing.
