Επιλέξτε μια πρόκληση

Το aMazeChallenge σάς επιτρέπει να ανταγωνίζεστε με τους φίλους σας και τους άλλους παίκτες στις προκλήσεις σε απευθείας σύνδεση. Κάντε κλικ σε μια πρόκληση από μια λίστα διαθέσιμων προκλήσεων για να την επιλέξετε.

Αφού επιλέξετε, θα μπορείτε να δείτε την τρέχουσα πρόκληση μαζί με όλους τους παίκτες που ανταγωνίζονται αυτήν τη στιγμή:

Δημιουργήστε τον κωδικό σας

Για να παίξετε online θα πρέπει να έχετε προετοιμάσει τον κωδικό σας εκ των προτέρων. Εάν όχι, μπορείτε να δημιουργήσετε ή να επεξεργαστείτε τον κώδικα στον επεξεργαστή κώδικα:

Ανεβάστε τον κωδικό σας

Αφού δημιουργήσετε ή επεξεργαστήκατε τον κωδικό σας, πρέπει να το μεταφορτώσετε:

Μετά την αποστολή του κωδικού σας, το avatar σας θα ξεκινήσει να εκτελεί τις εντολές σας, μόλις ξεκινήσει η πρόκληση. Για κάποιες προκλήσεις το παιχνίδι μπορεί να ξεκινήσει αμέσως.] Όταν η πρόκληση είναι πλήρης, θα τεθείτε σε αναμονή μέχρι κάποιος παίκτης να ολοκληρώσει την πρόκληση και να έρθει η σειρά σας.
