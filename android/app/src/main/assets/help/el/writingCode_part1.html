Γράφοντας κώδικα (1/3)

    

Οι συναρτήσεις

    

Υπάρχουν δύο συναρτήσεις που πρέπει να χρησιμοποιήσετε στον κώδικα:

    

Συνάρτηση προετοιμασίας (initialize)

    
    

Η συνάρτηση initialize εκτελεί τις οδηγίες της μόνο μία φορά και πριν εκτελεστεί οτιδήποτε άλλο. Χρησιμοποιείται για να δηλώσει μεταβλητές που θα χρησιμοποιηθούν στον υπόλοιπο κώδικα.

    

Συνάρτηση εκτέλεσης (run)

    
    

Η συνάρτηση εκτέλεσης (run) εκτελεί τις οδηγίες της συνεχώς (σε κάθε γύρο). Χρησιμοποιείται για να περιέχει τον αλγόριθμο που θα μετακινεί τον παίκτη μέσα στο λαβύρινθο.

    

Η συνάρτηση εκτέλεσης (run) πρέπει να είναι "συνδεδεμένη" με τη λειτουργία initialize. Μια παραδειγματική δομή κώδικα θα πρέπει να έχει την εξής μορφή:

    
    

Χρήσιμες συμβουλές

    
            
  • Μην τοποθετείτε κώδικα εκτός των συναρτήσεων προετοιμασίας (initialize) και εκτέλεσης (run). Δεν θα εκτελεστεί.
    •         
  • Προσπαθήστε να συμπεριλάβετε και τις δύο συναρτήσεις (αρχικοποίησης και εκτέλεσης) στον κώδικα σας. Η συνάρτηση εκτέλεσης (run) είναι υποχρεωτική. Η συνάρτηση αρχικοποίησης (initialize) μπορεί να παραλειφθεί.
    •         
  • Μην δηλώνετε μεταβλητές μέσα στη συνάρτηση εκτέλεσης. Αυτό θα επαναφέρει τις τιμές τους κάθε φορά που εκτελείται ο αλγόριθμος. Δηλώστε τις στην αρχικοποίηση (initialize).
    •         
  • Η δομή κάθε μπλοκ υποδεικνύει ποια είναι η εξαγωγή ή εισαγωγή του. Αν κάτι δεν ταιριάζει, τοτε δεν ήταν σκόπιμο να ταιριάζει.
    •         
  • Η αποθήκευση του κωδικού σας μετά από αλλαγές θα αποφύγει τυχόν απώλειες.
    •     
    
    

Kίνηση και Κατεύθυνση

    

Ο avatar σας μπορεί να εκτελέσει μία από τις παρακάτω ενέργειες σε κάθε γύρο. Όταν εκτελείται μία από αυτές τις εντολές, ολοκληρώνεται η σειρά σας.

    

Κίνηση προς τα εμπρός (Move Forward)

    

Αυτό θα προωθήσει το avatar σας ένα βήμα προς τα εμπρός.

    
    

Δεξιά Στροφή (Turn Clockwise)

    

Το avatar σας θα γυρίσει δεξιόστροφα σύμφωνα με την κατεύθυνση που βλέπει.

    
    

Αριστερή Στροφή

    

Το avatar σας θα γυρίσει αριστερόστροφα σύμφωνα με την κατεύθυνση που βλέπει.

    
    
    

Έλεγχος πιθανών κινήσεων

    

Οι ακόλουθες συναρτήσεις σάς επιτρέπουν να ελέγξετε αν το avatar σας μπορεί να μετακινηθεί προς μια συγκεκριμένη κατεύθυνση, ώστε να αποφύγετε να χάσετε τη σειρά σας. Θα επιστρέψουν true εάν ο avatar σας μπορεί να μετακινηθεί σε μια συγκεκριμένη θέση ή false αν δεν μπορεί.
