Δείγματα Κώδικα

Υπάρχουν τέσσερα δείγματα κώδικα που μπορείτε να χρησιμοποιήσετε για να μάθετε πώς να γράφετε κώδικα και να δείτε πώς λειτουργεί το παιχνίδι. Σε αυτό το παράδειγμα θα δούμε τον αλγόριθμο ακολουθίας αριστερού τοίχου (Left Wall Follower).

Ο ακολουθός αριστερού τοίχου είναι ένας βασικός αλγόριθμος επίλυσης λαβυρίνθου ο οποίος βασίζεται στη διατήρηση ενός "χεριού" στο αριστερό τοίχωμα του λαβύρινθου ανά πάσα στιγμή. Με απλά λόγια, ο avatar θα γυρίζει (αριστερά) όταν μπορεί, αλλιώς θα συνεχίσει να κινείται προς τα εμπρός. Τελικά, θα φτάσει στο τέλος δεδομένου ότι όλα τα τείχη στο λαβύρινθο είναι συνδεδεμένα.

Κατανόηση δείγματος κώδικα

Στο δείγμα κώδικα για τον Left Wall Follower, η μεταβλητή justTurned αρχικοποιείται στο false. Χρησιμοποιείται έτσι ώστε ο avatar να θυμάται αν έχει κάνει μια στροφή στον τελευταίο γύρο ή όχι. Ο αλγόριθμος λειτουργεί ως εξής:

        
  • Ο avatar θα κινηθεί προς τα εμπρός αν είχε γυρίσει (justTurned) κατά τη διάρκεια του τελευταίου γύρου ΚΑΙ μπορεί πράγματι να προχωρήσει ανάλογα με τη θέση και την κατεύθυνση του.
    •     
  • Εάν δεν μπορεί να μετακινηθεί προς τα εμπρός ή δεν γύρισε κατά τη διάρκεια του τελευταίου γύρου, θα επιχειρήσει να γυρίσει αριστερόστροφα (αν μπορεί να κινηθεί αριστερά).
    •     
  • Διαφορετικά, θα προσπαθήσει να προχωρήσει μπροστά (εάν μπορεί).
    •     
  • Εάν δεν μπορεί να γυρίσει προς τα αριστερά ή να προχωρήσει, θα γυρίσει προς τα δεξιά.

Ο avatar πρέπει να θυμάται αν γύρισε κατά τη διάρκεια του τελευταίου γύρου (ή όχι) έτσι ώστε να μην ξαναγυρίσει αν δεν χρειάζεται, διαφορετικά θα χανόταν.

Μπορείτε να επεξεργαστείτε δείγμα κώδικα όπως θέλετε για την παραγωγή καλύτερων αλγορίθμων τους οποίους θα αποθηκεύσετε στη συσκευή σας.
