Επιλογή πρόκλησης

Για να ξεκινήσετε ένα παιχνίδι, επιλέξτε μια πρόκληση από τη λίστα προκλήσεων. Θα μάθετε πώς να δημιουργήσετε τις δικές σας προκλήσεις (λαβύρινθους) αργότερα. Όλες οι διαθέσιμες προκλήσεις θα εμφανίζονται σε αυτήν τη λίστα. Επιλέξτε την πρώτη πρόκληση (Walk the line).

Ενδέχεται να σας ζητηθεί να γράψετε κώδικα εάν δεν το έχετε κάνει ήδη. Mεταβείτε στον Επεξεργαστή Κώδικα και εισαγάγετε τον ακόλουθο κώδικα:

        

Η επιλεγμένη πρόκληση (Walk the line) είναι να κίνηση σε ευθεία γραμμή. Για να γίνει αυτό, η μόνη απαιτούμενη κίνηση είναι να προχωρήσουμε απλά προς τα εμπρός μέχρι να φθάσουμε στο τέλος. Βεβαιωθείτε ότι έχετε συντάξει τον κωδικό σας, επιστρέψτε στην οθόνη εκπαίδευσης και επιλέξτε το 'Walk the line' ξανά. Θα πρέπει να δείτε την ακόλουθη οθόνη:

        
        

Αυτή η οθόνη δείχνει τη θέση σας στο λαβύρινθο με ένα βέλος χρωματισμένο με το επιλεγμένο χρώμα (και μια ετικέτα "You"). Η θέση εκκίνησης είναι χρωματισμένη με ανοιχτό κόκκινο χρώμα και η θέση τέλους είναι ανοιχτό πράσινο. Κανονικά, οι λαβύρινθοι θα έχουν τοίχους, όμως αυτός ο λαβύρινθος είναι άδειος.

        

Κάτω από τον λαβύρινθο μπορείτε να βρείτε την τρέχουσα υγεία και τους βαθμούς σας. Αυτά θα αλλάξουν κατά τη διάρκεια του παιχνιδιού. Πατήστε το κουμπί 'Επόμενο' μία φορά. Θα πρέπει να παρατηρήσετε ότι το avatar σας έχει προχωρήσει μία θέση μπροστά, όπως ακριβώς το έχετε προγραμματίσει. Μπορεί επίσης να έχετε παρατηρήσει διαφορετικά αντικείμενα που εμφανίζονται μέσα στο λαβύρινθο. Αυτά είναι τα αντικείμενα που περιγράψαμε νωρίτερα. Πατήστε ξανά το 'Επόμενο'.

        

Το avatar σας πρέπει τώρα να βρίσκεται στο μέσο του λαβύρινθου. Το κουμπί next πραγματοποιεί μία κίνηση κάθε φορά. Για να κάνετε συνεχόμενες κινήσεις, μπορείτε να πατήσετε το κουμπί Εκκίνηση/Παύση (αλλά μην το πατήσετε ακόμα). Το διάστημα (interval) θα καθορίσειπόσο γρήγορα θα εκτελεστούν οι κινήσεις (το λιγότερο είναι ταχύτερο). Μπορείτε να πατήσετε 'Λεπτομέρειες' για να δείτε στατιστικά για το τρέχον παιχνίδι.

        

Πατήστε το κουμπί 'Εκκίνηση' για να ολοκληρώσετε το λαβύρινθο.
