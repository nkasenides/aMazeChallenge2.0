Σχεδιαστής λαβύρινθων

Δημιουργία πρόκλησης

Μπορείτε να χρησιμοποιήσετε το Σχεδιαστή Λαβύρινθων για να δημιουργήσετε τις δικές σας προκλήσεις.

Δώστε ένα όνομα (μόνο γράμματα και αριθμοί, χωρίς κενά) και μια περιγραφή για την πρόκλησή σας. Ο επιλεγμένος αλγόριθμος θα καθορίσει εάν η πρόκλησή σας θα έχει μία ή περισσότερες λύσεις ή εάν θα έχει μικρό αριθμό τοίχων ή καθόλου τοίχους.

Επιλέξτε το μέγεθος του λαβυρίνθου σας και τις θέσεις εκκίνησης και τερματισμού. Οι αριθμοί ξεκινούν από το επάνω αριστερό μέρος και τελειώνουν κάτω δεξιά. Για παράδειγμα, μια θέση εκκίνησης με η σειρά = 0 και η στήλη = 0 θα είναι στην επάνω αριστερή πλευρά και μια θέση στόχου με τη σειρά = 4 και τη στήλη = 4 θα είναι στην κάτω δεξιά.

Μπορείτε επίσης να επιλέξετε τον αριθμό των ανταμοιβών και των ποινών που επιβάλλονται στον λαβύρινθο σας. Ο ορισμός των τιμών σε Χαμηλή θα έχει μικρότερο από το επιλεγμένο είδος και η ρύθμιση τους σε Υψηλή θα παράγει περισσότερα. Εναλλακτικά μπορείτε να αποφύγετε κυρώσεις ή εμπόδια επιλέγοντας «κανένα».

Κάτω από αυτές τις επιλογές μπορείτε να επιλέξετε το χρώμα τοίχου του λαβυρίνθου και την εικόνα φόντου που θέλετε να χρησιμοποιήσετε από μια προκαθορισμένη λίστα. Βεβαιωθείτε ότι χρησιμοποιείτε χρώματα που αντιτίθενται μεταξύ τους διαφορετικά τα τοιχώματα των λαβυρίνθων δεν θα είναι ορατά.

Τέλος, μπορείτε επίσης να επιλέξετε έναν ήχο για τον λαβύρινθο που βασίζεται στο θέμα του από μια προκαθορισμένη λίστα.

Όταν τελειώσετε, κάντε κλικ στο κουμπί "Δημιουργία". Στη συνέχεια, μπορείτε να προσθέσετε τον παραγόμενο λαβύρινθο στην εκπαίδευση.
